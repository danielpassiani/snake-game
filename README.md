# Jogo da Cobrinha (Snake Game)

Versão simples do clássico Jogo da Cobrinha feita com HTML5, CSS e JavaScript puro.

## Como rodar

1. Abra o arquivo `index.html` no navegador (duplo clique) ou use um servidor local.

```powershell
start index.html
```

2. Controle com as setas do teclado. Placar aumenta +10 por comida.

## Recursos

- Canvas 400x400
- Controles com as setas (impede reversão instantânea)
- Placar, crescimento da cobrinha, colisões e reinício automático
- Controles de velocidade, pausar/continuar e sons básicos
